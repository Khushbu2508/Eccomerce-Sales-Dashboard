***Eccomerce-Sales-Dashboard***

**Project Overview**
- Interactive E-Commerce Sales Dashboard designed to analyze business performance
- Focuses on sales, profit, quantity, and customer behavior
- Helps stakeholders make data-driven decisions using visual insights

Key KPIs
- Total Sales (Amount)
- Total Quantity Sold
- Total Profit
- Average Order Value (AOV)

 Time-Based Analysis
- Profit analysis by month
- Quarterly performance comparison
- Seasonal trend identification

Product & Category Insights
- Quantity distribution by category:
Clothing ,Electronics,Furniture
- Profit by sub-category (Printers, Bookcases, Saree, Accessories, Tables)
- Identification of high-performing and low-performing categories

Payment Mode Analysis
COD (Cash on Delivery),UPI,Credit Card,Debit Card,EMI

Customer Insights
- Sales contribution by top customers
- Identification of high-value customers

Tools & Technologies Used
- Power BI / Microsoft Excel
- Data Cleaning & Transformation
- Data Modeling
- Data Visualization Techniques
- Business Analytics Concepts
